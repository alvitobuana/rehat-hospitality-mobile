# Attendance Feature Placeholder

Direktori ini disiapkan sebagai wadah modularisasi untuk fitur absensi karyawan (Attendance) di Sprint berikutnya.

## Komponen Mendatang:
- `attendance_screen.dart` (Halaman Absensi Masuk / Pulang)
- `attendance_provider.dart` (Riverpod State Notifier untuk absensi)
- `attendance_repository.dart` (Integrasi HTTP API absensi)
